## 1. 数据库优化设计

### 1.1 索引设计

SQLite 索引策略（基于热榜/情感/日志三大表查询特征）：

| 表名 | 索引名 | 字段 | 类型 | 说明 |
|------|--------|------|------|------|
| hot_topics | `idx_hot_topics_platform` | `platform_id` | B-tree | 按平台筛选 |
| hot_topics | `idx_hot_topics_crawl_time` | `crawl_time` | B-tree | 按时间范围查询 |
| hot_topics | `idx_hot_topics_heat_score` | `heat_score` | B-tree | 按热度排序 |
| hot_topics | `idx_hot_topics_category` | `category` | B-tree | 按分类筛选 |
| sentiment_results | `idx_sentiment_label` | `sentiment_label` | B-tree | 按情感标签筛选 |
| sentiment_results | `idx_sentiment_analyzed_at` | `analyzed_at` | B-tree | 按分析时间排序 |
| crawl_logs | `idx_crawl_logs_platform` | `platform_id` | B-tree | 按平台查询日志 |
| crawl_logs | `idx_crawl_logs_started_at` | `started_at` | B-tree | 按时间查询日志 |
| crawl_logs | `idx_crawl_logs_status` | `status` | B-tree | 按状态统计 |

#### 创建索引的 SQL

```sql
-- hot_topics 表索引
CREATE INDEX IF NOT EXISTS idx_hot_topics_platform ON hot_topics(platform_id);
CREATE INDEX IF NOT EXISTS idx_hot_topics_crawl_time ON hot_topics(crawl_time);
CREATE INDEX IF NOT EXISTS idx_hot_topics_heat_score ON hot_topics(heat_score DESC);
CREATE INDEX IF NOT EXISTS idx_hot_topics_category ON hot_topics(category);

-- sentiment_results 表索引
CREATE INDEX IF NOT EXISTS idx_sentiment_label ON sentiment_results(sentiment_label);
CREATE INDEX IF NOT EXISTS idx_sentiment_analyzed_at ON sentiment_results(analyzed_at DESC);

-- crawl_logs 表索引
CREATE INDEX IF NOT EXISTS idx_crawl_logs_platform ON crawl_logs(platform_id);
CREATE INDEX IF NOT EXISTS idx_crawl_logs_started_at ON crawl_logs(started_at DESC);
CREATE INDEX IF NOT EXISTS idx_crawl_logs_status ON crawl_logs(status);
```

#### 复合索引（覆盖高频查询）

```sql
-- 平台+时间（Dashboard 热榜列表）
CREATE INDEX IF NOT EXISTS idx_hot_topics_platform_crawl_time

-- 平台+状态（采集成功率统计）
CREATE INDEX IF NOT EXISTS idx_crawl_logs_platform_status

```

### 1.2 查询优化

#### 分页查询优化

SQLite 大数据量分页的 OFFSET 性能问题：

```sql
-- 低效：OFFSET 越大越慢
SELECT * FROM hot_topics ORDER BY crawl_time DESC LIMIT 20 OFFSET 10000;

-- 高效：使用游标分页（基于时间戳）
SELECT * FROM hot_topics 
WHERE crawl_time < '2026-07-07 10:00:00' 
ORDER BY crawl_time DESC LIMIT 20;
```

#### 统计查询优化

```sql
-- 今日热榜数（使用日期函数索引）
SELECT COUNT(*) FROM hot_topics 
WHERE date(crawl_time) = date('now');

-- 优化：使用范围查询（命中索引）
SELECT COUNT(*) FROM hot_topics 
WHERE crawl_time >= date('now') AND crawl_time < date('now', '+1 day');
```

## 2. 数据分析算法

### 2.1 描述统计分析

对热榜数据的描述性统计指标：

| 指标 | 计算方式 | 用途 |
|------|----------|------|
| 均值热度 | `AVG(heat_score)` | 平台平均热度水平 |
| 中位数热度 | 排序后中间值 | 排除极端值影响的典型热度 |
| 标准差 | `STD(heat_score)` | 热度波动程度 |
| 最大值/最小值 | `MAX/MIN(heat_score)` | 热度范围 |
| 分位数 | 25%/50%/75% | 热度分布特征 |
| 变异系数 | `CV = std / mean` | 热度稳定性 |

#### SQL 实现

```sql
-- 平台热度描述统计
SELECT 
    p.name as platform,
    COUNT(*) as topic_count,
    AVG(ht.heat_score) as avg_heat,
    MAX(ht.heat_score) as max_heat,
    MIN(ht.heat_score) as min_heat,
    (SELECT heat_score FROM hot_topics 
     WHERE platform_id = p.id 
     ORDER BY heat_score DESC 
     LIMIT 1 OFFSET (SELECT COUNT(*) FROM hot_topics WHERE platform_id = p.id) / 2
    ) as median_heat
FROM platforms p
JOIN hot_topics ht ON p.id = ht.platform_id
WHERE ht.crawl_time >= date('now', '-7 days')
GROUP BY p.id;
```

#### Python 实现（Pandas）

```python
import pandas as pd
import numpy as np

def analyze_heat_distribution(topics_df: pd.DataFrame) -> dict:
    """
    热度分布分析
    
    Args:
        topics_df: 热榜数据 DataFrame (含 heat_score, platform_id)
        
    Returns:
        dict: 各平台统计指标
    """
    stats = topics_df.groupby('platform_id')['heat_score'].agg([
        ('count', 'count'),
        ('mean', 'mean'),
        ('std', 'std'),
        ('min', 'min'),
        ('max', 'max'),
        ('median', 'median'),
        ('q25', lambda x: x.quantile(0.25)),
        ('q75', lambda x: x.quantile(0.75)),
    ]).round(2)
    
    # 计算变异系数
    stats['cv'] = (stats['std'] / stats['mean']).round(2)
    
    return stats.to_dict()
```

### 2.2 趋势分析算法

#### 时间序列分析

```python
from datetime import datetime, timedelta
import pandas as pd
from typing import List, Dict

def analyze_trend(
    topics_df: pd.DataFrame,
    freq: str = 'D',  # 'H' 小时, 'D' 天, 'W' 周
    metric: str = 'count'  # 'count', 'avg_heat', 'max_heat'
) -> List[Dict]:
    """
    时间序列趋势分析
    
    Args:
        topics_df: 热榜数据 DataFrame
        freq: 聚合频率
        metric: 统计指标
        
    Returns:
        List[Dict]: 时间序列数据
    """
    # 设置时间索引
    df = topics_df.copy()
    df['crawl_time'] = pd.to_datetime(df['crawl_time'])
    df.set_index('crawl_time', inplace=True)
    
    if metric == 'count':
        result = df.resample(freq).size()
    elif metric == 'avg_heat':
        result = df.resample(freq)['heat_score'].mean()
    elif metric == 'max_heat':
        result = df.resample(freq)['heat_score'].max()
    else:
        result = df.resample(freq).size()
    
    return [
        {"date": str(date), "value": round(float(val), 2) if not pd.isna(val) else 0}
        for date, val in result.items()
    ]


def detect_trend_change(series: List[float]) -> Dict:
    """
    检测趋势突变点
    
    使用简单的一阶差分检测变化率超过阈值的点
    
    Args:
        series: 时间序列值列表
        
    Returns:
        Dict: 突变点索引及变化率
    """
    if len(series) < 3:
        return {"changes": []}
    
    changes = []
    for i in range(1, len(series)):
        if series[i-1] == 0:
            continue
        change_rate = (series[i] - series[i-1]) / series[i-1]
        if abs(change_rate) > 0.5:  # 变化率超过 50%
            changes.append({
                "index": i,
                "change_rate": round(change_rate, 2),
                "direction": "up" if change_rate > 0 else "down"
            })
    
    return {"changes": changes}
```

#### 环比/同比分析

```python
def calculate_growth_rate(
    current_value: float,
    previous_value: float
) -> float:
    """
    计算增长率
    
    Args:
        current_value: 当前值
        previous_value: 上期值
        
    Returns:
        float: 增长率（百分比）
    """
    if previous_value == 0:
        return 0.0
    return round((current_value - previous_value) / previous_value * 100, 2)


def period_over_period_analysis(
    df: pd.DataFrame,
    period: str = 'day'  # 'day', 'week', 'month'
) -> Dict:
    """
    环比分析
    
    Args:
        df: 热榜数据 DataFrame
        period: 对比周期
        
    Returns:
        Dict: 环比指标
    """
    df['crawl_time'] = pd.to_datetime(df['crawl_time'])
    
    if period == 'day':
        current = df[df['crawl_time'].dt.date == pd.Timestamp.now().date()]
        previous = df[df['crawl_time'].dt.date == (pd.Timestamp.now() - pd.Timedelta(days=1)).date()]
    elif period == 'week':
        current = df[df['crawl_time'] >= pd.Timestamp.now() - pd.Timedelta(weeks=1)]
        previous = df[
            (df['crawl_time'] >= pd.Timestamp.now() - pd.Timedelta(weeks=2)) &
            (df['crawl_time'] < pd.Timestamp.now() - pd.Timedelta(weeks=1))
        ]
    else:  # month
        current = df[df['crawl_time'].dt.month == pd.Timestamp.now().month]
        previous = df[df['crawl_time'].dt.month == (pd.Timestamp.now() - pd.Timedelta(days=30)).month]
    
    return {
        "current_topics": len(current),
        "previous_topics": len(previous),
        "topic_growth": calculate_growth_rate(len(current), len(previous)),
        "current_avg_heat": current['heat_score'].mean() if len(current) > 0 else 0,
        "previous_avg_heat": previous['heat_score'].mean() if len(previous) > 0 else 0,
        "heat_growth": calculate_growth_rate(
            current['heat_score'].mean() if len(current) > 0 else 0,
            previous['heat_score'].mean() if len(previous) > 0 else 0
        ),
    }
```

### 2.3 关联分析

#### 平台间热度关联

```python
import pandas as pd
from typing import Dict, List

def analyze_platform_correlation(topics_df: pd.DataFrame) -> Dict:
    """
    分析各平台热度相关性
    
    Args:
        topics_df: 热榜数据 DataFrame (含 platform_id, heat_score, crawl_time)
        
    Returns:
        Dict: 相关系数矩阵
    """
    # 按平台和日期聚合
    df = topics_df.copy()
    df['date'] = pd.to_datetime(df['crawl_time']).dt.date
    
    # 透视表：行为日期，列为平台，值为平均热度
    pivot = df.groupby(['date', 'platform_id'])['heat_score'].mean().unstack(fill_value=0)
    
    # 计算相关系数矩阵
    corr_matrix = pivot.corr().round(2)
    
    return {
        "correlation_matrix": corr_matrix.to_dict(),
        "strongest_pair": corr_matrix.stack().sort_values(ascending=False).iloc[1],  # 排除对角线
        "weakest_pair": corr_matrix.stack().sort_values().iloc[0],
    }


def find_cross_platform_hot_topics(
    topics_df: pd.DataFrame,
    time_window: int = 24  # 小时
) -> List[Dict]:
    """
    查找跨平台同时出现的热门话题（共同话题）
    
    Args:
        topics_df: 热榜数据 DataFrame
        time_window: 时间窗口（小时）
        
    Returns:
        List[Dict]: 共同话题列表
    """
    # 使用文本相似度匹配（简化版：标题包含相同关键词）
    from difflib import SequenceMatcher
    
    df = topics_df.copy()
    df['crawl_time'] = pd.to_datetime(df['crawl_time'])
    
    # 按时间窗口分组
    df['time_window'] = df['crawl_time'].dt.floor(f'{time_window}H')
    
    cross_platform = []
    
    for window, group in df.groupby('time_window'):
        titles = group[['title', 'platform_id']].to_dict('records')
        
        for i in range(len(titles)):
            for j in range(i+1, len(titles)):
                similarity = SequenceMatcher(None, titles[i]['title'], titles[j]['title']).ratio()
                if similarity > 0.6:  # 相似度阈值
                    cross_platform.append({
                        "time_window": str(window),
                        "title_1": titles[i]['title'],
                        "platform_1": titles[i]['platform_id'],
                        "title_2": titles[j]['title'],
                        "platform_2": titles[j]['platform_id'],
                        "similarity": round(similarity, 2)
                    })
    
    return cross_platform
```

### 2.4 情感分布分析

```python
from collections import Counter
from typing import Dict, List

def analyze_sentiment_distribution(
    sentiment_results: List[Dict]
) -> Dict:
    """
    情感分布深度分析
    
    Args:
        sentiment_results: 情感分析结果列表
        
    Returns:
        Dict: 多维度情感分析
    """
    labels = [r['sentiment_label'] for r in sentiment_results]
    confidences = [r['confidence'] for r in sentiment_results]
    
    total = len(labels)
    counter = Counter(labels)
    
    distribution = {
        label: {
            "count": count,
            "percentage": round(count / total * 100, 2)
        }
        for label, count in counter.items()
    }
    
    # 置信度分析
    high_conf = sum(1 for c in confidences if c > 0.8)
    
    return {
        "total": total,
        "distribution": distribution,
        "confidence_analysis": {
            "high_confidence_ratio": round(high_conf / total * 100, 2) if total > 0 else 0,
            "avg_confidence": round(sum(confidences) / len(confidences), 2) if confidences else 0,
        },
        "dominant_sentiment": counter.most_common(1)[0][0] if counter else None,
    }


def sentiment_time_series(
    sentiment_df: pd.DataFrame,
    freq: str = 'D'
) -> List[Dict]:
    """
    情感趋势时间序列
    
    Args:
        sentiment_df: 情感分析结果 DataFrame
        freq: 聚合频率
        
    Returns:
        List[Dict]: 各时间点的情感分布
    """
    df = sentiment_df.copy()
    df['analyzed_at'] = pd.to_datetime(df['analyzed_at'])
    df.set_index('analyzed_at', inplace=True)
    
    # 按时间和标签分组计数
    result = df.groupby([pd.Grouper(freq=freq), 'sentiment_label']).size().unstack(fill_value=0)
    
    return [
        {
            "date": str(date),
            "positive": int(row.get('positive', 0)),
            "negative": int(row.get('negative', 0)),
            "neutral": int(row.get('neutral', 0)),
        }
        for date, row in result.iterrows()
    ]
```

## 3. 数据挖掘算法

### 3.1 文本聚类（K-Means + TF-IDF）

对热榜话题进行自动聚类，发现潜在话题群组。

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.cluster import KMeans
from sklearn.decomposition import PCA
import pandas as pd
import numpy as np
from typing import List, Dict

def cluster_topics(
    topics: List[str],
    n_clusters: int = 5,
    max_features: int = 1000
) -> Dict:
    """
    K-Means 文本聚类
    
    Args:
        topics: 话题标题列表
        n_clusters: 聚类数量
        max_features: TF-IDF 最大特征数
        
    Returns:
        Dict: 聚类结果
    """
    if len(topics) < n_clusters:
        return {"clusters": [], "message": "话题数量不足"}
    
    # 1. TF-IDF 向量化
    vectorizer = TfidfVectorizer(
        max_features=max_features,
        stop_words='english',  # 可替换为中文停用词
        ngram_range=(1, 2),    # 1-gram 和 2-gram
        min_df=2,              # 忽略低频词
        max_df=0.8,            # 忽略高频词
    )
    
    tfidf_matrix = vectorizer.fit_transform(topics)
    
    # 2. K-Means 聚类
    kmeans = KMeans(
        n_clusters=n_clusters,
        random_state=42,
        n_init=10,
    )
    labels = kmeans.fit_predict(tfidf_matrix)
    
    # 3. 提取每个簇的关键词
    feature_names = vectorizer.get_feature_names_out()
    cluster_keywords = []
    
    for i in range(n_clusters):
        # 获取簇中心向量
        center = kmeans.cluster_centers_[i]
        # 取权重最高的 10 个词
        top_indices = center.argsort()[-10:][::-1]
        keywords = [feature_names[j] for j in top_indices]
        cluster_keywords.append(keywords)
    
    # 4. 组装结果
    clusters = []
    for i in range(n_clusters):
        cluster_topics_list = [topics[j] for j in range(len(topics)) if labels[j] == i]
        clusters.append({
            "cluster_id": i,
            "size": len(cluster_topics_list),
            "keywords": cluster_keywords[i],
            "sample_topics": cluster_topics_list[:5],
        })
    
    # 5. PCA 降维（用于可视化）
    pca = PCA(n_components=2)
    coordinates = pca.fit_transform(tfidf_matrix.toarray())
    
    return {
        "clusters": clusters,
        "coordinates": [
            {"x": float(x), "y": float(y), "cluster": int(label)}
            for (x, y), label in zip(coordinates, labels)
        ],
        "n_clusters": n_clusters,
    }
```

### 3.2 异常检测（Isolation Forest）

检测异常热度（突然爆火或突然消失的话题）。

```python
from sklearn.ensemble import IsolationForest
import pandas as pd
import numpy as np
from typing import List, Dict

def detect_anomalous_topics(
    topics_df: pd.DataFrame,
    contamination: float = 0.05
) -> List[Dict]:
    """
    使用 Isolation Forest 检测异常热度话题

        topics_df: 热榜数据 DataFrame (需含 heat_score, crawl_time 等)
        contamination: 异常比例（默认 5%）
        
    Returns:
        List[Dict]: 异常话题列表
    """
    df = topics_df.copy()
    
    # 特征工程
    df['crawl_time'] = pd.to_datetime(df['crawl_time'])
    df['hour'] = df['crawl_time'].dt.hour
    df['day_of_week'] = df['crawl_time'].dt.dayofweek
    
    # 特征向量
    features = df[['heat_score', 'hour', 'day_of_week']].fillna(0)
    
    # 标准化
    from sklearn.preprocessing import StandardScaler
    scaler = StandardScaler()
    scaled_features = scaler.fit_transform(features)
    
    # 训练异常检测模型
    model = IsolationForest(
        contamination=contamination,
        random_state=42,
        n_estimators=100,
    )
    predictions = model.fit_predict(scaled_features)
    scores = model.decision_function(scaled_features)
    
    # 异常样本 (prediction == -1)
    anomalies = df[predictions == -1].copy()
    anomalies['anomaly_score'] = scores[predictions == -1]
    
    return [
        {
            "topic_id": row['topic_id'],
            "title": row['title'],
            "heat_score": row['heat_score'],
            "platform_id": row['platform_id'],
            "anomaly_score": round(row['anomaly_score'], 4),
            "reason": "heat_score anomaly" if row['heat_score'] > df['heat_score'].quantile(0.95) else "pattern anomaly",
        }
        for _, row in anomalies.iterrows()
    ]


def detect_heat_spike(
    topic_history: List[Dict],
    threshold: float = 3.0
) -> Dict:
    """
    检测热度突增（基于 Z-Score）
    
    Args:
        topic_history: 话题历史热度数据 [{time, heat_score}, ...]
        threshold: Z-Score 阈值
        
    Returns:
        Dict: 突增检测结果
    """
    if len(topic_history) < 3:
        return {"has_spike": False, "reason": "数据不足"}
    
    heat_scores = [h['heat_score'] for h in topic_history]
    mean = np.mean(heat_scores[:-1])  # 排除最新值
    std = np.std(heat_scores[:-1])
    
    if std == 0:
        return {"has_spike": False, "reason": "无变化"}
    
    latest = heat_scores[-1]
    z_score = (latest - mean) / std
    
    return {
        "has_spike": z_score > threshold,
        "z_score": round(z_score, 2),
        "latest_heat": latest,
        "avg_heat": round(mean, 2),
        "threshold": threshold,
    }
```

```python
def extract_keywords_tfidf(

    Args:

        all_text,

        withWeight=True,
        allowPOS=('ns', 'n', 'vn', 'v', 'nr')  # 地名、名词、动名词、动词、人名
    )
    
    return [(word, round(weight, 4)) for word, weight in keywords]


def extract_keywords_textrank(
    texts: List[str],
    top_k: int = 20
) -> List[Tuple[str, float]]:
    """
    使用 TextRank 提取关键词
    
    TextRank 基于图排序，不需要语料库，适合单文档关键词提取
    
    Args:
        texts: 文本列表
        top_k: 提取关键词数量
        
    Returns:
        List[Tuple[str, float]]: (关键词, 权重) 列表
    """
    all_text = ' '.join(texts)
    
    keywords = jieba.analyse.textrank(
        all_text,
        topK=top_k,
        withWeight=True,
        allowPOS=('ns', 'n', 'vn', 'v', 'nr')
    )
    
    return [(word, round(weight, 4)) for word, weight in keywords]


def extract_platform_keywords(
    topics_df: pd.DataFrame,
    top_k: int = 10
) -> Dict[str, List[Tuple[str, float]]]:
    """
    按平台提取关键词
    
    Args:
        topics_df: 热榜数据 DataFrame
        top_k: 每个平台提取关键词数量
```

### 3.4 话题生命周期分析

```python
from datetime import datetime, timedelta
from typing import List, Dict, Optional

def analyze_topic_lifecycle(
    topic_history: List[Dict]
) -> Dict:
    """
    分析话题生命周期（萌芽 → 爆发 → 成熟 → 衰退）
    
    Args:
        topic_history: 话题历史数据 [{time, heat_score}, ...]
        
    Returns:
        Dict: 生命周期阶段分析
    """
    if len(topic_history) < 3:
        return {"stage": "unknown", "reason": "数据不足"}
    
    # 按时间排序
    history = sorted(topic_history, key=lambda x: x['time'])
    heat_scores = [h['heat_score'] for h in history]
    
    # 计算变化率
    changes = []
    for i in range(1, len(heat_scores)):
        if heat_scores[i-1] == 0:
            changes.append(0)
        else:
            changes.append((heat_scores[i] - heat_scores[i-1]) / heat_scores[i-1])
    
    # 判断阶段
    latest_change = changes[-1] if changes else 0
    avg_change = sum(changes) / len(changes) if changes else 0
    max_heat = max(heat_scores)
    latest_heat = heat_scores[-1]
    
    # 阶段判断逻辑
    if latest_change > 0.3 and latest_heat < max_heat * 0.5:
        stage = "emerging"  # 萌芽：快速上升但总量不高
    elif latest_change > 0.1 and latest_heat >= max_heat * 0.7:
        stage = "peak"  # 爆发：高热度且仍在上升
    elif abs(latest_change) < 0.1 and latest_heat >= max_heat * 0.5:
        stage = "mature"  # 成熟：高热度但趋于平稳
    elif latest_change < -0.2:
        stage = "declining"  # 衰退：快速下降
    else:
        stage = "stable"  # 稳定
    
    return {
        "stage": stage,
        "latest_change_rate": round(latest_change, 2),
        "avg_change_rate": round(avg_change, 2),
        "max_heat": max_heat,
        "latest_heat": latest_heat,
        "data_points": len(history),
        "stage_description": {
            "emerging": "话题热度快速上升，处于爆发前期",
            "peak": "话题热度达到顶峰，关注度最高",
            "mature": "话题热度维持高位，趋于稳定",
            "declining": "话题热度快速下降，关注度减退",
            "stable": "话题热度平稳，无显著变化",
        }.get(stage, "未知阶段"),
    }
```

### 4.2 模型微调方案

```python
from transformers import (
    BertTokenizer,
    BertForSequenceClassification,
    Trainer,
    TrainingArguments,
    DataCollatorWithPadding,
)
from datasets import Dataset
import torch
import numpy as np
from sklearn.model_selection import train_test_split
from typing import List, Dict

class SentimentModel:
    """
    情感分析模型封装
    
    支持：
    1. 预训练模型加载
    2. 微调训练
    3. 批量推理
    4. 模型评估
    """
    
    def __init__(self, model_name: str = "hfl/chinese-roberta-wwm-ext", num_labels: int = 3):
        """
        初始化模型
        
        Args:
            model_name: HuggingFace 模型名称
            num_labels: 分类数量（3=正面/负面/中性）
        """
        self.model_name = model_name
        self.num_labels = num_labels
        self.device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
        
        # 加载 tokenizer 和模型
        self.tokenizer = BertTokenizer.from_pretrained(model_name)
        self.model = BertForSequenceClassification.from_pretrained(
            model_name,
            num_labels=num_labels,
        ).to(self.device)
        
        # 标签映射
        self.label_map = {0: "negative", 1: "neutral", 2: "positive"}
        self.id_map = {v: k for k, v in self.label_map.items()}
    
    def predict(self, texts: List[str], batch_size: int = 32) -> List[Dict]:
        """
        批量预测
        
        Args:
            texts: 待分析文本列表
            batch_size: 批大小
            
        Returns:
            List[Dict]: 预测结果
        """
        results = []
        
        for i in range(0, len(texts), batch_size):
            batch = texts[i:i+batch_size]
            
            # Tokenize
            inputs = self.tokenizer(
                batch,
                padding=True,
                truncation=True,
                max_length=512,
                return_tensors="pt",
            ).to(self.device)
            
            # 推理
            with torch.no_grad():
                outputs = self.model(**inputs)
                probabilities = torch.softmax(outputs.logits, dim=-1)
                predictions = torch.argmax(probabilities, dim=-1)
            
            # 组装结果
            for j, text in enumerate(batch):
                pred_id = predictions[j].item()
                probs = probabilities[j].cpu().numpy()
                
                results.append({
                    "text": text,
                    "sentiment_label": self.label_map[pred_id],
                    "confidence": round(float(probs[pred_id]), 4),
                    "scores": {
                        "negative": round(float(probs[self.id_map["negative"]]), 4),
                        "neutral": round(float(probs[self.id_map["neutral"]]), 4),
                        "positive": round(float(probs[self.id_map["positive"]]), 4),
                    },
                })
        
        return results
    
    def fine_tune(
        self,
        train_texts: List[str],
        train_labels: List[str],
        val_texts: List[str] = None,
        val_labels: List[str] = None,
        epochs: int = 3,
        batch_size: int = 16,
        learning_rate: float = 2e-5,
        output_dir: str = "./model_output",
    ):
        """
        微调模型
        
        Args:
            train_texts: 训练文本
            train_labels: 训练标签 ("positive" / "negative" / "neutral")
            val_texts: 验证文本
            val_labels: 验证标签
            epochs: 训练轮数
            batch_size: 批大小
            learning_rate: 学习率
            output_dir: 输出目录
        """
        # 转换标签为 ID
        train_label_ids = [self.id_map[label] for label in train_labels]
        
        # 创建 Dataset
        train_data = Dataset.from_dict({
            "text": train_texts,
            "label": train_label_ids,
        })
        
        # Tokenize
        def tokenize_function(examples):
            return self.tokenizer(
                examples["text"],
                padding=True,
                truncation=True,
                max_length=512,
            )
        
        train_data = train_data.map(tokenize_function, batched=True)
        
        # 训练参数
        training_args = TrainingArguments(
            output_dir=output_dir,
            num_train_epochs=epochs,
            per_device_train_batch_size=batch_size,
            per_device_eval_batch_size=batch_size,
            learning_rate=learning_rate,
            weight_decay=0.01,
            logging_dir="./logs",
            logging_steps=10,
            evaluation_strategy="epoch" if val_texts else "no",
            save_strategy="epoch",
            load_best_model_at_end=True if val_texts else False,
        )
        
        # 数据整理器
        data_collator = DataCollatorWithPadding(tokenizer=self.tokenizer)
        
        # 训练器
        trainer = Trainer(
            model=self.model,
            args=training_args,
            train_dataset=train_data,
            data_collator=data_collator,
        )
        
        # 训练
        trainer.train()
        
        # 保存模型
        self.model.save_pretrained(output_dir)
        self.tokenizer.save_pretrained(output_dir)
        print(f"Model saved to {output_dir}")
    
    def evaluate(self, texts: List[str], labels: List[str]) -> Dict:
        """
        评估模型
        
        Args:
            texts: 测试文本
            labels: 真实标签
            
        Returns:
            Dict: 评估指标
        """
        from sklearn.metrics import accuracy_score, precision_recall_fscore_support, confusion_matrix
        
        predictions = self.predict(texts)
        pred_labels = [p["sentiment_label"] for p in predictions]
        
        # 计算指标
        accuracy = accuracy_score(labels, pred_labels)
        precision, recall, f1, _ = precision_recall_fscore_support(
            labels, pred_labels, average="weighted"
        )
        
        # 混淆矩阵
        cm = confusion_matrix(labels, pred_labels, labels=["negative", "neutral", "positive"])
        
        return {
            "accuracy": round(accuracy, 4),
            "precision": round(precision, 4),
            "recall": round(recall, 4),
            "f1_score": round(f1, 4),
            "confusion_matrix": cm.tolist(),
        }
```

```python
class OptimizedSentimentModel:
    """
    生产环境优化版情感分析模型
    
    优化点：
    1. 模型量化（INT8）减少内存占用
    2. 批量推理提高吞吐
    3. 缓存常用结果
    4. 异步推理
    """
    
    def __init__(self, model_path: str, use_quantization: bool = True):
        self.device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
        
        # 加载模型
        self.model = BertForSequenceClassification.from_pretrained(model_path)
        self.tokenizer = BertTokenizer.from_pretrained(model_path)
        
        # 量化优化（CPU 环境）
        if use_quantization and self.device.type == "cpu":
            self.model = torch.quantization.quantize_dynamic(
                self.model,
                {torch.nn.Linear},
                dtype=torch.qint8,
            )
            print("Model quantized to INT8")
        
        self.model.to(self.device)
        self.model.eval()
        
        # 结果缓存
        self.cache = {}
        self.cache_size = 1000
    
    def predict_with_cache(self, text: str) -> Dict:
        """带缓存的预测"""
        # 检查缓存
        cache_key = hash(text)
        if cache_key in self.cache:
            return self.cache[cache_key]
        
        # 预测
        result = self.predict([text])[0]
        
        # 存入缓存（LRU 简单实现）
        if len(self.cache) >= self.cache_size:
            self.cache.pop(next(iter(self.cache)))
        self.cache[cache_key] = result
        
        return result
```
